# Parseable on Railway

One-click deploy Parseable log analytics on Railway.

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template?template=https://github.com/YOUR_USERNAME/parseable-railway)

## Environment Variables

| Variable | Value |
|---|---|
| P_USERNAME | admin |
| P_PASSWORD | your_password |
| P_STAGING_DIR | /tmp/parseable/staging |
| P_HOT_TIER_PATH | /var/lib/parseable |

## Usage

After deploy, open the Railway URL in browser and login with your credentials.
